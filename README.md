#### 发布步骤看录屏
1.注册账号
2.点击File->Open Scene->选择文件->Import
3.PC端直接File->Build Settings->选择PC->Build
4.安卓端要先点Swich Platform和Player Settings改Company和Other Settings里的Bundle Identifier再选择Build就可以了
